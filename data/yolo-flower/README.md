# 花卉识别数据集 (Flower Recognition Dataset)

这是一个用于花卉识别的YOLO格式数据集，包含102种不同的花卉种类。

## 数据集介绍

- **类别数量**: 102种花卉
- **图片总数**: 8188张图片
- **数据格式**: YOLO格式（图片+标签）
- **数据集划分**:
  - 训练集 (train): 约6550张图片 (80%)
  - 验证集 (val): 约819张图片 (10%)
  - 测试集 (test): 约819张图片 (10%)

## 数据集结构

```
yolo_dataset/
  ├── data.yaml        # 配置文件
  ├── classes.txt      # 类别名称文件
  ├── images/
  │    ├── train/      # 训练集图片
  │    ├── val/        # 验证集图片
  │    └── test/       # 测试集图片
  └── labels/
       ├── train/      # 训练集标签
       ├── val/        # 验证集标签
       └── test/       # 测试集标签
```

## 类别列表

数据集包含103种花卉，详细列表请参考`classes.txt`文件。部分常见花卉包括：
- 玫瑰 (rose)
- 向日葵 (sunflower)
- 郁金香 (tulip)
- 兰花 (orchid)
- 百合 (lily)
等。

## 数据集处理

本数据集是通过处理原始花卉数据集创建的，所有图片已按照YOLO标准格式进行了整理和标注。图片文件按照`[花卉英文名]_[序号].jpg`的格式命名，标签文件则对应为`[花卉英文名]_[序号].txt`。

## 注意事项

- 所有标签文件采用YOLO格式，每行的格式为：`[类别ID] [中心点x] [中心点y] [宽度] [高度]`
- 所有坐标值都已归一化到0-1之间
- 建议在训练时使用数据增强以提高模型泛化能力
